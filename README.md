#  Machine Learning Lab Programs

This repository contains all the Machine Learning programs and experiments performed during my lab sessions.

##  About

The programs in this repository are part of my academic curriculum and focus on understanding the fundamentals of Machine Learning, data analysis, and model building.

##  Topics Covered

* Data Preprocessing
* Data Visualization
* Supervised Learning

  * Linear Regression
  * Logistic Regression
  * Decision Trees
  * K-Nearest Neighbors (KNN)
* Unsupervised Learning

  * K-Means Clustering
  * K-Medoid Clustering 
* Model Evaluation Techniques

## Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib / Seaborn
* Scikit-learn

## How to Run

1. Install required libraries:

   ```
   pip install numpy pandas matplotlib seaborn scikit-learn
   ```
2. Clone this repository:

   ```
   git clone https://github.com/kprem20/IML-LAB-SEM-4th
   ```
3. Run any program:

   ```
   python filename.py
   ```
